## 2026-03-11（モックアップ作成セッション）

### やったこと
- 全画面HTMLモックアップを作成（30画面、単一HTML）
- コラボハウスのブランドカラーをロゴSVGから抽出して適用（オレンジ#F99900、チャコール#302D2C、オリーブ#574E2E）
- GitHub Pagesでモックアップを公開（https://n-matsusaka.github.io/hr-evaluation/）
- レスポンシブ対応（ハンバーガーメニュー、モバイルレイアウト）

### 決定事項

**UI/UX**
- 等級表記は「Lv.4-2」形式（レベルとステップを結合、Lvとステップは別カラムにしない）
- サイドバーはチャコール背景にブランドオレンジのアクセント
- ダッシュボードは3種類（被評価者/評価者/管理者）をロールで切り替え
- 被評価者ダッシュボードはグラフィカルに（ドーナツチャート、レーダーチャート、棒グラフ）
- 評価プロセスは5ステップステッパー（目標設定→評価入力→レビュー→確定→公開）で表示
- ブラウザタブタイトルは「CH評価」、ファビコンはオレンジ背景に白の「CH」

**面談記録**
- 年度単位で管理（年度タブで切り替え）
- 面談カードクリックで面談詳細画面に遷移
- 面談詳細は構造化表示（基本情報・内容・次回予定・過去履歴）

**総評コメント**
- コメント者情報を表示（アバター・名前・役割・日付）
- 1次評価者（オレンジボーダー）、2次評価者（ブルーボーダー）、来期の期待（グリーンボーダー）で色分け

**社員管理**
- 社員一覧→社員詳細は動的切り替え（全職種対応）
- 社員詳細画面はプロフィール・評価履歴・能力推移チャートで構成

**設定画面**
- 7タブ構成: 会社情報・職種マスタ・レベルステップ・評価ウェイト・通知設定・権限管理・プロフィール
- 評価ウェイト: 成果40%/能力35%/行動25%
- Leader Radar係数: S=0.5〜1.5(手動)/A=0.4/B=0.3/C=0.2/D=0.0
- 通知: 6種類（期間開始・目標締切・評価締切・360度依頼・確定通知・面談記録）
- 権限: 管理者/評価者/一般社員 × 10機能のマトリクス
- SmartHR連携URLの設定欄を用意
